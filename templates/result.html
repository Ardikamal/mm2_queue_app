Hasil Perhitungan M/M/2

Input

  • Waktu antar kedatangan = {{ arrival_time }} menit
  • Waktu pelayanan per pelayan = {{ service_time }} menit

Langkah Perhitungan

1. Laju Kedatangan (λ)

λ = 1 / waktu antar kedatangan = 1 / {{ arrival_time }} = {{ "%.4f"|format(lam) }}

2. Laju Pelayanan (μ)

μ = 1 / waktu pelayanan = 1 / {{ service_time }} = {{ "%.4f"|format(mu) }}

3. Pemanfaatan Pelayan (ρ)

ρ = λ / (2μ) = {{ "%.4f"|format(lam) }} / (2 × {{ "%.4f"|format(mu) }}) = {{ "%.4f"|format(rho) }}

4. Waktu Rata-rata dalam Sistem (W)

W = 1 / (μ − λ/2) = {{ "%.4f"|format(W) }} menit

5. Waktu Rata-rata dalam Antrian (Wq)

Wq = λ² / [2μ(μ − λ/2)] = {{ "%.4f"|format(Wq) }} menit

Kesimpulan

  • Rata-rata pelanggan berada dalam sistem selama {{ "%.2f"|format(W) }} menit
  • Rata-rata waktu menunggu di antrian adalah {{ "%.2f"|format(Wq) }} menit
